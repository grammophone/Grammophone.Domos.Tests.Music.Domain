# Grammophone.Domos.Tests.Music.Domain

Music-domain entities used by the Domos integration test application.

This project defines the fictional music catalog, record label, album, track, user, disposition, and workflow transition entities used to exercise Domos entity security and workflow security without depending on a private application domain.

## Target Frameworks

- `net472`
- `netstandard2.1`

## Required Projects

This project expects these sibling projects to be available when building from the solution or from extracted submodules:

Direct project references:

- `Grammophone.Domos.Domain`
- `Grammophone.GenericContentModel`

Additional transitive project references:

- None
